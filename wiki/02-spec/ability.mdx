# Ability

An ability is a collection of [blocks](/wiki/spec/block) that are always
referenced by [rules](/wiki/spec/rule) and [loop blocks](/wiki/spec/block#loops).


## Metadata

### `abilityID`

<PropertyData type="string" />

The ability's unique identifier, referenced in
[rules](/wiki/spec/rule#abilityid) and [loop blocks](/wiki/spec/block#controlscript).

```json
"abilityID": "561D32F8-FD6B-45E0-9DFF-3191085E2351-3248-00005BE3B1E67098"
```


### [`blocks`](/wiki/spec/block)

<PropertyData type="array" />

Contains [block](/wiki/spec/block) objects in the ability.

```json
"blocks": []
```


### `createdAt`

<PropertyData type="number" />

The creation timestamp of the ability, representing the number of seconds since
January 1, 2001, 00:00 UTC.

Used to order custom blocks in the app's block picker.

```json
"createdAt": 528311636.47971
```


## Metadata (Custom Block)

### `name`

<PropertyData type="string" />

The display name of a custom block.

If empty, a custom block will not appear in the app's block picker.
Blocks with identical names can coexist without issues.

```json
"name": "Do a Cool Trick"
```
