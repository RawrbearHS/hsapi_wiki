# Block

Blocks are objects listed inside of [abilities](/wiki/spec/ability).
They are also included in the [parameter](/wiki/spec/parameter)'s [`datum`](/wiki/spec/parameter#datum)
property, with some differences.


## Common Properties

### `block_class`

<PropertyData type="string" />

*Unused.* Represents the block's class type. Possible values include:<Note note="needs elaboration" />

- `"method"`: for loopless blocks, such as Move Forward, Set Variable, Comment, etc.
- `"conditionalControl"`: for Check Once and Check If Else loops.
- `"control"`: for Draw a Trail and other loop blocks.<Note note="needs verification" />
- `"conditionalOperator"`: for conditional operators, including `=`, `≠`, `<` and `>`.
- `"operator"`: for math and string operators, such as `+`, `-`, `×`, `÷`, Random, and HSB.

```json
"block_class": "conditionalControl"
```


### `type`

<PropertyData type="number" />
